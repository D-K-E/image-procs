Scattering

Problem Statement

What happens when light hits on the surface of an object ?

Description

When thinking about light scattering, it is better to let the model guide the discourse rather than the inverse. Light scattering is modeled as bidirectional reflectance distribution function ( BRDF ). It describes how much wavelength incoming to a relative surface coordinate is emitted. The function can be written as \( f_r(\theta_{\text{incoming}}, \phi_{\text{incoming}}, \theta_{\text{emitted}}, \phi_{\text{emitted}}; \lambda) \) where \(\phi\) and \( \theta \) are spherical coordinates for the incident flux, meaning they are the angles in which the flux is arriving and emitted.

It is a reciprocal function where you can change the roles of the incoming and emitted angles and you would obtain the same value anyway due to the physics of light transport. The full model has two parts: \[ L_{r}(\hat{v};\lambda) = \int L_i(\theta_i; \lambda) \hspace{0.5em} f_r(\theta_i, \phi_i, \theta_e, \phi_e; \lambda)\hspace{0.5em} max(0, \cos(\theta_i))\hspace{0.5em} d \theta_i \phi_i \]

Phong Shading Model: \[ I_{\lambda} = I_{a\lambda}k_a O_{d\lambda} + f_{att}I_{p\lambda}[k_d O_{d\lambda} (\hat{N} \cdot \hat{L}) + k_s (\hat{R} \cdot \hat{V})^n] \hspace{1em} \hat{R} \cdot \hat{V} = (2\hat{N} (\hat{N} \cdot \hat{L}) - \hat{L} ) \cdot \hat{V} \]

Here is the components of the model
\(I_{\lambda}\)
Light intensity for channel \(\lambda\). In practice this is the value for one of your color channels R,G, or B. In most cases it is a normalized value between 0 and 1.0, we interpolate it to a value between 0 255.
\(I_{a\lambda}\)
Ambient light intensity for the channel \(\lambda\). The amount of light in the environment.
\(k_a\)
Ambient light coefficient. This help us to control how much the ambient light should effect the overall color of the object.
\(O_{d\lambda}\)
Object diffuse coefficient. It is a material property of the object. It models how much the object reacts to the wavelength \(\lambda\) of the light.
f_{att}
Attenuation value. Since light reduces its intensity as it travels from its caster to the material that scatters it. If this would not be the case, we would not be able to distinguish objects that emit same color and that are equidistant to us. Their distance to the light source changes the attenuation factor of the light rays that scatter on them, which in turn reflects as an intensity change, that intensity change help us to distinguish the objects. In most cases it is calculated as follows: \( f_att = \frac{1}{a_c + a_l * d + a_q * d^2} \) where \(a_c\) is constant factor, \(a_l\) is linear factor, \(a_q\) is quadratic factor.
