Coordinates

Problem Statement

How do you designate the location of something ?

Description

Coordinates in 1 Dimensional Space

When we would like to refer to the location of a point in \(\mathbb{R}\) line, we can simply use its relation to other numbers on the line, if \(x_i < x\) then the x is on the right of \(x_i\), if \(x_i > x\) then the x is on the left of \(x_i\). The distance between x and \(x_i\) is simply \({\Delta}x = |x - x_i|\).

Coordinates in 2 Dimensional Space
Cartesian Coordinates

A point P in \(\mathbb{R}^2\) is expressed with two real numbers as \(P = (a, b)\). Now let us draw a horizontal line, x and a vertical line, y, which intersect at a point O with \(x = \{ x_i | x_i \in \mathbb{R} \} \) and \(y = \{y_j | y_j \in \mathbb{R} \}\) where \( i, j \in \mathbb{N}\). The projection of P onto x is the number a, whereas the projection of P onto y is the number b. The projection \(p: \mathbb{R} \to \mathbb{R} \) for any \(a_i, b_i \in \mathbb{R}\) is:

injective
that is for all pairs that are in its domain there exist a single value in its codomain
surjective
that is all values in its codomain has a correspondant in its domain.

The collection of all pairs that join the domain and the codomain of such projection constitutes the 2 dimensional euclidean plane and the representation that we use is known as cartesian coordinate system. The main advantage of using cartesian coordinate system is that it permit us to refer to a component of point as a function of the other one.

Polar Coordinates

Given a circle whose center is at O and a horizontal line that passes through the center a point, in 2 dimensional euclidean plane can also be expressed with the point's distance and the angle to the center of the circle.

Elementary Operations

Problem Statement

What are necessary operations that one needs to know in pre-calculus in order to compute values ?

Operator List

Addition:

Description
Mostly used with infix operator \(+\). When two or more disjoint sets combined the number of elements in the resulting set are represented with addition.
Commutative
a + b = b + a, meaning order of elements does not change result
Associative
a + (b + c) = (a + b) + c meaning the precedence of operations does not change the final result
Identity Element
a + 0 = a, meaning that 0 is a number which does not change the identity of the element.
Units
in the operation a + b we assume that a and b have the same units, for example 4m + 5m = 9m but 4m + \(5m^2\) is not possible.

Subtraction:

Description
It is usually designated with the infix operator \(-\). It represents going back some steps in the number line.
Anticommutative
a - b = - ( b - a) the result is the negative of the original result
Non Associative
\(a - (b - c) \not = (a - b) - c\) the precedence of operations contributes to the result

Multiplication:

Description
Usually designated by the infix operator \(\times\) or \(\cdot\). It can be thought of as a repeated addition.
Commutative
a × b = b × a
Associative
a × (b × c) = (a × b) × c
Distributive
a × (b + c) = a × b + a × c
Identity element
a × 1 = a
Absorbing element
a × 0 = 0
Negation
-1 × a = -a that is multiplication with -1 results in additive inverse
Inverse Element
every number except 0 has an multiplicative inverse so that \(a × 1/a = 1\)
Order preservation
multiplication by positive number conserves order, by a negative number reverses order: \(a \times b < a \times c if b < c -a \times b > -a \times c if b < c \)

Division:

Description
Usually designated by the infix operator \(/\), and \(÷\). It can be considered the process of calculating how many times a number is contained in another one.
Distributive
(a+b) / c = a/c + b/c

Exponentiation

Description
It is usually designated with a base and power part as in \(b^n\) where b is base and n is power. It represents the repeated multiplication of base like \(b^n = b \times b \times ... n times ... \times b\)
Identities
\(b^{m+n} = b^m \times b^n\)
\(b^{n} \times b = b^{n+1}\)
\(b \times c)^n = b^n \times c^n\)
\( b^{c^n} = b^{n \times c}\)
Zero Exponent
\(b^0 = 1\)
\(b^{n} \times b = b^{n+1}
Negative Exponent
\(b^{-n} = 1/b^{-n}\)
Positive Rational Exponent
\( b^{m/n} = b^{{1/n}^m} = \sqrt[n]{b^m} = \sqrt[n]{b}^m \)

Logarithm:

Description
Logarithm is the inverse of exponential function. For \(b^y = x, \log_b(x) = y\), basically I have the base and the resulting value I try to find the power.
Product
\(\log_b(a × c) = \log_b(a) + \log_b(c) \)
Division
\(\log_b(x / y ) = \log_b(x) - \log_b(y) \)
Exponent
\( \log_b(x^p) = p \log_b(x) \)
Root
\( \log_b(\sqrt[p]{x}) = \log_b(x) / p \)
Change of Base
\( \log_b(x) = \log_k(x) / \log_k(b) \)

Infinite Numerical Sequences

Problem Statement

I have a lot of pieces that are very very very very small. If I start counting them, I might not be able to finish before I die, but I also want to express that they are countable. Plus, what if the pieces are infinitely small, or what if pieces are big enough but there is no end to them ? How do I describe this situation

Description

Definition of an infinite numerical sequence by Tarasov 1982 is:

We say that there is an infinite numerical sequence if every natural number (position number) is unambiguously placed in correspondence with a definite number (term of the sequence) by a specific rule
Here are some examples:
  1. \( 1, 2, 4, 8, 16, ... \)
  2. \( 1/2, 3/4, 5/6, 7/8, ... \)
  3. \( 1, 1/2, 1/3, 1/4, ... \)
  4. \( 1, 1/2, 3, 1/4, 5, 1/6, ... \)
  5. \( 1, -1, 1/3, -1/3, 1/5, -1/5, 1/7, -1/7, ... \)
Note how definition applies to these sequences for example: \( [1 \to ] 1 ,[2 \to ] 2, [3 \to ] 4, [4 \to ] 8, ... [n \to ] 2^{n-1}\) Thus the sequence can also be represented more formally as \([1 \to ] y_1 ,[2 \to ] y_2, [3 \to ] y_3, [4 \to ] y_4, ... [n \to ] y_n\)
